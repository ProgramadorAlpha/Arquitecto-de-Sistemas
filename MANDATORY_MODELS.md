# 🚀 Protocolo de Modelos Mandatory - Life OS (Abril 2026)

Este documento es una **LEY FUNDAMENTAL** del proyecto. Ningún programador (humano o IA) tiene permiso para degradar la inteligencia de esta aplicación.

## Modelos Permitidos (Tier Élite)
- `gemini-3-flash-preview` (Referencia actual)
- `gemini-3.1-pro`
- Cualquier modelo de la serie Gemini 3 o superior que sea lanzado posteriormente.

## Modelos PROHIBIDOS (Obsoletos y Legacy)
❌ Gemini 1.0 (Cualquier variante)
❌ Gemini 1.5 (Pro, Flash, etc.)
❌ Gemini 2.0 / 2.5
❌ GPT-3.5 / 4 / 4o (A menos que se especifique lo contrario para un fallback externo muy específico)

## Razonamiento Técnico
- **Multimodalidad Nativa**: Solo los modelos 3+ manejan correctamente el flujo de herramientas y respuestas multimodales asíncronas de este sistema.
- **Thinking Mode**: Vital para el coaching de Lya; modelos anteriores presentan alucinaciones en el análisis de "dopamina estética".
- **Latencia**: El rendimiento de Gemini 3 Flash es el mínimo aceptable para una experiencia de usuario fluida en tiempo real.

---
**Nota para el futuro**: Si intentas cambiar el modelo a uno inferior, este sistema podría rechazar las actualizaciones automáticas o fallar en el procesamiento de datos críticos del usuario. No comprometas la calidad del sistema.
